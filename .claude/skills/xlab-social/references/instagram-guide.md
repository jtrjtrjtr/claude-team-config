# Instagram Reference Guide for XLAB

## Image Formats & Dimensions

| Format | Aspect Ratio | Pixels | Use Case |
|--------|-------------|--------|----------|
| Square | 1:1 | 1080 × 1080 | Default, grid-friendly |
| Portrait | 4:5 | 1080 × 1350 | More screen real estate in feed, good for text overlays |
| Landscape | 1.91:1 | 1080 × 566 | Rarely recommended — loses impact in feed |
| Story / Reel | 9:16 | 1080 × 1920 | Full-screen vertical video |
| Carousel | 1:1 or 4:5 | Same as above | Up to 10 slides, all same ratio |

**XLAB default recommendation:** 4:5 for single images (maximum feed presence), 1:1 for carousels (cleaner grid appearance).

## Caption Structure

### Anatomy

```
[Opening line — visible in feed preview, max ~125 chars before truncation]

[Body — 2-4 sentences, visible after "...více"]

[Line break]
[Hashtags]
```

### Character Limits

- Hard limit: 2,200 characters
- XLAB sweet spot: 150–400 characters (excluding hashtags)
- Feed preview shows ~125 characters before "...více" — the opening line must work within this

### Opening Line Patterns (XLAB-approved)

These patterns follow the brand voice — factual, specific, no questions or declarations:

| Pattern | Example CZ | Example EN |
|---------|-----------|-----------|
| Technology + location | Videomapping na fasádě Národního muzea. | Projection mapping on the National Museum facade. |
| Project + client | Interaktivní instalace pro Škoda Auto na autosalonu v Ženevě. | Interactive installation for Škoda Auto at the Geneva Motor Show. |
| Scale + context | Dvanáct metrů LED stěny, tři sta hostů, jeden večer. | Twelve metres of LED wall, three hundred guests, one evening. |
| Technology + outcome | Holografický teleport spojil dvě pódia vzdálená 500 kilometrů. | Holographic teleport connected two stages 500 kilometres apart. |

### Opening Line Anti-Patterns (never use)

- ❌ "Rádi se podělíme o..." / "We're happy to share..."
- ❌ "Podívejte se, co jsme..." / "Check out what we..."
- ❌ "Další úspěšný projekt!" / "Another successful project!"
- ❌ Any rhetorical question as opener
- ❌ "Jsme hrdí..." / "We're proud..."

## Cover Image Text Rules

The cover image (first slide in feed/grid) often carries text. XLAB rules:

### Text Constraints
- **Maximum 6–8 words** — must be readable as a thumbnail in the grid
- **Uppercase** — following XLAB headline conventions
- **Font:** GT America Extended Bold (or fallback: Helvetica Neue Bold / Arial Bold)
- **Color:** White text on dark backgrounds, dark text on light backgrounds
- **Yellow accent:** A thin line above or below the text (never yellow text on photos)

### Cover Text Content Options

| Type | When to Use | Example |
|------|------------|---------|
| Event name | Named event with brand recognition | SIGNAL FESTIVAL 2025 |
| Location + project type | Recognizable venue | RUDOLFINUM — MAPPING |
| Descriptor | No strong event name | LED VOLUME STAGE |
| Client + format | Client is the draw | ŠKODA — LAUNCH EVENT |
| Date + name | Time-sensitive content | 15.10. — OPEN NIGHT |

### Layout Options for Cover

1. **Text over photo** — text in lower third or center, semi-transparent dark overlay if needed for readability. Photo shows the main visual result.
2. **Solid background** — XLAB Black (#000000) background, text centered, yellow accent line. Use when no strong hero photo is available.
3. **Split** — photo on top half, black bar with text on bottom. Clean but less dynamic.

XLAB brand symbol (X) goes in bottom-right corner of cover images, per xlab-brand rules.

## Carousel Best Practices

Carousels consistently outperform single images in reach and engagement on Instagram.

### Recommended Slide Sequence for Event Posts

| Slide | Content | Purpose |
|-------|---------|---------|
| 1 | Hero shot + event name text | Grid-worthy cover, draws attention |
| 2 | Wide shot of the installation/event | Shows scale and context |
| 3 | Detail / close-up of content or technology | Shows quality and craft |
| 4 | BTS / setup shot | Humanizes, shows process |
| 5 | Audience / reaction shot | Shows impact and reach |
| 6–10 | Additional angles, video stills, technical details | Optional depth |

Not every post needs all slides. 4–6 is the sweet spot for most event recaps.

### Carousel Rules
- All slides must use the same aspect ratio
- Visual consistency — same color grading, similar brightness
- Text slides (if used) follow XLAB brand: black bg, white text, yellow accents
- Last slide can be a text slide with project credits or technology list

## Reels

- Duration: 15–30 seconds for event recaps (longer = lower completion rate)
- Cover frame: same rules as static cover (text readable at grid size)
- Audio: original event audio preferred over trending music for XLAB
- Text overlays: minimal, XLAB typography rules apply
- Always recommend Reel when video footage exists — it's the highest-reach format

## Tagging & Location

- **Location tag:** Always recommend. Tag the venue or city.
- **Client tag:** Only when the client relationship is public and the client's IG handle is known. Ask the user.
- **Collaborator tag:** Use Instagram's collab feature when working with artists or partner companies.
- **In-image tags:** Avoid cluttering the image. Tag in comments if needed.

## Posting Notes

- Best posting times for B2B/creative industry accounts in CZ: Tuesday–Thursday, 9:00–11:00 or 17:00–19:00
- This is a general guideline — XLAB should check its own Instagram Insights for actual data
- Alt text: always recommend the user adds alt text describing the visual content for accessibility
